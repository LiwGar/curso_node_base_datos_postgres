## Running in your local machine
Clone the repository:
```sh
git@github.com:LiwGar/course_node_api_rest_express.git
```
Move to the api folder:
```sh
cd node_express
```
Install dependencies:
```sh
npm i
```
Run it!
```sh
npm run start
```
