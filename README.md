## Running in your local machine
Clone the repository:
```sh
git@github.com:LiwGar/curso_node_base_datos_postgres.git
```
Move to the api folder:
```sh
cd node_express
```
Install dependencies:
```sh
npm i
```
Run it!
```sh
npm run start
```
